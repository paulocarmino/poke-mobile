# Tecnologias envolvidaas

- React Native
- GraphQL
- Styled Components
- Expo

# Rodando o projeto

```
# Clone esse repositorio
git clone https://github.com/paulocarmino/poke-mobile

# Entre no repositorio
cd poke-mobile

# Instale as dependencias
yarn install

# Rode o app
yarn run start
```
